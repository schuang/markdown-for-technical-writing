Markdown example

Lists

It's very easy to make some words bold and other words italic with Markdown. You can even link to Google!

  • The first important point
    • sub-item 1
    • sub-item 2
  • The second important point

Alternatively,

  • The first important point
  • The second important point

Numbered lists are also supported:

  1. First point
  2. Second point

Links

URL is automatically recognized:

http://github.com

Use this syntax to control the displayed text:

GitHub

Code

Python code block with syntax highlighting:

def foo():
    sum = 0
    for i in [1,2,3,4]:
        sum += i
    return sum
int main()
{
    const int N = 10;
    int sum = 0;
    for (int i=0; i<N; i++) {
        sum += i;
    }
    printf("sum = %d\n", sum);
    return 0;
}
